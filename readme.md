# DataTerrain: 3D-Printable Spatial Histograms

**DataTerrain** is a specialized Python-based workflow and application built with **Streamlit** designed to transform geographic boundary maps and tabular datasets into physical, 3D-printable spatial histograms. 

Instead of just looking at flat colors on a screen, DataTerrain parses, simplifies, and maps metrics (like election turnouts, population density, or economic data) directly to physical elevation heights. It optimizes the raw geographic shapes to generate clean, sliceable boundaries ready to be exported to an `.stl` for manufacturing or 3D printing.
